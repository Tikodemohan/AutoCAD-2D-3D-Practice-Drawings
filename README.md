# 📐 AutoCAD 2D & 3D Practice Drawings

## Overview

This repository contains a collection of AutoCAD 2D and 3D practice drawings along with project drawings. The exercises are designed to improve CAD drafting skills, technical drawing accuracy, and practical knowledge of engineering drawings.

## Objective

The objective of this project is to practice AutoCAD 2D and 3D drafting through a variety of engineering drawing exercises and project-based examples.

## Key Features

- 2D drawing exercises
- 3D drawing exercises
- Engineering project drawings
- Beginner-friendly practice material
- PDF format for easy reference

## Software Used

- AutoCAD

## Learning Outcomes

- Improved 2D and 3D drafting skills
- Better understanding of AutoCAD tools and commands
- Enhanced engineering drawing accuracy
- Increased confidence in creating technical drawings

## Conclusion

This project demonstrates practical AutoCAD drafting skills through a collection of 2D drawings, 3D drawings, and engineering projects. It is suitable for students and beginners who want to strengthen their CAD design skills.
